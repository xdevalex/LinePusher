# LP-CB-07 - Push History Access

## Summary

The user can access historical notification records via an external link.

---

## Trigger

- The user selects the "Push History" menu action.

---

## Expected Behavior

- The system provides a URL linking to notification history.
- Notification history is viewed outside of LINE.

---

## Invariants

- LINE does not display detailed notification logs inline.
- Accessing history does not affect notification delivery or quota.

---

## Non-Goals

- Defining history retention period or storage details.
- Defining filtering, searching, or export capabilities.
